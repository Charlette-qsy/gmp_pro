### Mask Editor Standard

+ Edit Box item

每一个编辑框需要带一个文本框说明参数的含义。文本框位于编辑框的上面。文本框的文字内容末尾没有标点，需要包含参数的物理意义和单位；编辑框的提示符内容需要写为`参数名称:`。
